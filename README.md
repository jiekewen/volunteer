# 高考志愿填报 Prompt 生成器

一个纯前端的单页工具。输入高考信息后，自动生成带真实录取数据的 Prompt，复制到任意 AI 即可获得志愿填报分析。

## 使用方法

1. 双击 `index.html` 在浏览器中打开
2. 填写高考信息（省份、选科、分数、偏好等）
3. 点击生成 Prompt
4. 一键复制，粘贴到 DeepSeek / Kimi / 豆包 等 AI 中
