## 🔧 如何设置自动发货（买家拍下自动发送）

闲鱼本身不支持全自动发货，但有以下几种方式：

### 方案一：闲鱼「快捷回复」（推荐，最简单的半自动）

1. 打开闲鱼 App → 我的 → 设置 → 快捷回复
2. 添加一条快捷回复，内容就是上面的「自动回复话术」
3. 有人拍下后，打开对话，点快捷回复 → 一键发送

只要设置一次，每次都秒发。

### 方案二：自动回复机器人（真正的全自动）

1. 在微信搜「闲鱼自动发货」小程序
2. 或在淘宝搜「闲鱼自动回复」「闲鱼发货助手」
3. 购买一个自动发货服务（通常按月收费，几块钱一个月）
4. 按说明绑定闲鱼账号，设置自动回复内容
5. 买家拍下后自动发送消息

### 方案三：把链接写在详情页（零成本但不推荐）

直接在商品描述里放链接，买家拍下后自己看描述就行。但容易被平台判定为站外引流，有下架风险。

## ⚠️ 闲鱼避坑指南

1. **不要出现「微信」「QQ」等站外联系方式** — 会被系统检测下架
2. **不要卖违规内容** — 虚拟商品描述尽量写「教程」「指南」「资料」而非「软件」
3. **及时回复** — 闲鱼回复率影响排名，尽量半小时内回复
4. **引导好评** — 买家满意后礼貌请对方确认收货 + 好评，好评多了排名才靠前
5. **多上几个链接** — 用不同标题和主图同时上 3-5 个链接，测试哪个标题点击率高
6. **每天擦亮宝贝** — 在闲鱼「我发布的」里点「擦亮」，每天可免费获得曝光
