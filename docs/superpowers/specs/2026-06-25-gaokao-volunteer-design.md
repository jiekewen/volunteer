# 高考志愿填报 Prompt 生成器 — 设计文档

## 概述

纯前端单页工具。用户输入高考分数、省份、选科类型等个人信息后，工具自动加载该省份的真实录取数据，拼装成一份结构化的超详细 Prompt。用户一键复制后，可粘贴到 ChatGPT、Claude、Gemini 等任意 AI 模型中获取志愿填报建议。

## 技术选型

| 项目 | 选择 | 原因 |
|------|------|------|
| 框架 | 零框架，单 HTML + CSS + JS | 体量小，无需构建，双击即用 |
| 数据 | 按省份拆分 JSON，fetch 按需加载 | 避免一次性加载全部数据 |
| 部署 | GitHub Pages | 免费，零成本，全球可访问 |
| 存储 | localStorage | 保存填写历史，刷新不丢失 |

## 覆盖省份

首批 7 个省市：江苏、河南、河北、上海、北京、天津、黑龙江

## 用户输入

### 必填（硬指标）
- 省份（下拉选择）
- 选科类型（物理类 / 历史类）
- 高考分数（手动输入）
- 批次线（优先从数据文件自动填入，支持手动修改）

### 选填（偏好标签，多选）
- 地域倾向：省内优先 / 可以出省 / 一线城市优先 / 不限
- 学校层次：985 / 211 / 双一流 / 公办本科 / 不限
- 专业方向：工科 / 医学 / 师范 / 经管 / 法学 / 理学 / 农学 / 文学 / 不限
- 发展倾向：优先就业 / 优先考研 / 优先考公 / 出国深造

### 选填（自由描述）
- 文本框，用于补充标签无法覆盖的个人偏好

## Prompt 生成逻辑

1. 根据用户选科类型匹配对应科目组的专业数据
2. 按 分数 - 专业最低分 的差值分三档：
   - 冲刺档：差值在 [-20, -5) 区间
   - 稳妥档：差值在 [-5, +10) 区间
   - 保底档：差值在 [+10, +40] 区间
3. 每档按分数降序排列，取前若干条
4. 将筛选结果 + 用户偏好标签 + 自由描述 填入 Prompt 模板

## Prompt 模板结构

```
# 高考志愿填报分析请求

## 1. 考生画像
- 省份、选科类型、分数、批次线、大致位次

## 2. 真实录取数据参考（以下为{省份}{年份}年实际录取数据）
### 🚀 冲刺档（{N}所院校 {M}个专业）
| 院校 | 专业 | 最低分 | 平均分 | 招生人数 |
|------|------|--------|--------|----------|
| ...  | ...  | ...    | ...    | ...      |

### ✅ 稳妥档（{N}所院校 {M}个专业）
...

### 🛡️ 保底档（{N}所院校 {M}个专业）
...

## 3. 考生偏好
- 标签：...
- 个人描述：...

## 4. 分析输出要求
请基于以上真实数据和考生偏好，输出以下内容：

a) **冲/稳/保三档推荐表**
   - 推荐院校 + 专业组合
   - 每个组合标注：录取概率评估（低/中/高）、与去年分数差值
   - 专业实力简评（学科评估等级、是否特色专业）

b) **专业就业前景分析**
   - 每个推荐专业的就业方向（毕业后主要去向、典型岗位）
   - 行业发展趋势（未来 3-5 年就业市场预测）
   - 薪资水平参考（应届生起薪范围）

c) **个性化建议**
   - 结合考生的标签偏好和自由描述，给出针对性推荐
   - 如果偏好之间存在矛盾（如"想留省内但分数够不到省内好学校"），指出并提供折中方案

d) **风险提示与备选方案**
   - 冲刺档的风险点（大小年、专业调剂风险）
   - 如果滑档的应急方案（征集志愿、下一批次优质选择）
```
